## Usage

Most scripts can be run with `tsx` (TypeScript) or `node` (JavaScript):

```bash
# TypeScript scripts
npx tsx scripts/database/check-balances.ts

# JavaScript scripts
node scripts/analysis/calc-sum-sicredi.js

# Python scripts
python scripts/import/add_nanoid_to_csv.py
```
